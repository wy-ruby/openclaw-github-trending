# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-10

### Added
- Initial release
- GitHub trending fetching (daily, weekly, monthly)
- AI-powered repository summaries (OpenAI, Anthropic)
- Multi-channel push support (Feishu, Email)
- Smart deduplication with star growth tracking
- OpenClaw scheduled task integration
- History management and persistence
- Comprehensive configuration schema
- Full TypeScript support
- Unit tests with >80% coverage

### Security
- Secure API key storage in OpenClaw config
- Support for app-specific email passwords
- Input validation and sanitization

## [Unreleased]

### Planned
- Support for more AI providers (Claude, Gemini)
- Additional push channels (Slack, Discord, Telegram)
- Customizable summary templates
- Repository filtering by language or topic
- Web dashboard for history viewing
